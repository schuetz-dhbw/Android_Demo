# Android Demo App

Example application used in the Android Development lecture at DHBW Heidenheim to demonstrate the basic concepts for the introduction. 

## Concepts covered
- Multi-screen navigation with Jetpack Compose Navigation 3
- MVVM architecture with ViewModel and StateFlow
- Dependency injection with Hilt
- Local persistence with Room
- Remote data fetching with Retrofit (Random User API)
- DataStore for user preferences

## Tech stack
Kotlin, Jetpack Compose, Hilt, Room, Retrofit, DataStore
